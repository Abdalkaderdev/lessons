# Omar's Curriculum

## Goal
Learn to use AI tools (Claude) to build and deploy real websites using modern tools.

## Modules

### Module 1: Getting Started with Claude
- What is AI and how Claude works
- How to write good prompts
- Using Claude to help with coding
- **Lesson:** [Lesson 01 - Intro to Claude](lesson-01-intro-to-claude.md)

### Module 2: GitHub Basics
- What is GitHub and why developers use it
- Creating an account and your first repository
- Basic Git commands (clone, add, commit, push)
- **Lesson:** [Lesson 02 - GitHub Basics](lesson-02-github-basics.md)

### Module 3: Building Your First Website
- HTML & CSS basics
- Using Claude to generate and fix code
- Building a simple personal page
- **Lesson:** [Lesson 03 - Build a Website](lesson-03-build-a-website.md)

### Module 4: Deploying with Vercel
- What is deployment and hosting
- Connecting GitHub to Vercel
- Deploying your site with one click
- **Lesson:** [Lesson 04 - Vercel Deployment](lesson-04-vercel-deployment.md)

### Module 5: Domains
- What is a domain name and how DNS works
- Free vs paid domains
- Connecting a custom domain to Vercel
- **Lesson:** [Lesson 05 - Domains](lesson-05-domains.md)

### Module 6: Next Steps
- Adding JavaScript interactivity
- Exploring Next.js
- Building a portfolio site
