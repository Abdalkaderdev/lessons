## 1. What is AI?

AI (Artificial Intelligence) is software that can understand language, generate text, write code, and solve problems. Think of it as a very smart assistant that learned from reading billions of pages of text.

**Key point:** AI doesn't "think" like humans — it predicts the best response based on patterns it learned.

## 3. Writing Good Prompts

A **prompt** is what you type to Claude. Better prompts = better answers.

### Bad Prompt
> "make me a website"

### Good Prompt
> "Create a simple HTML page for a personal portfolio. Include my name 'Omar', a short bio section, and links to my GitHub and social media. Use modern CSS with a dark theme."

### Tips for Good Prompts
1. **Be specific** — say exactly what you want
2. **Give context** — explain what it's for
3. **Ask step by step** — break big tasks into smaller ones
4. **Include examples** — show what you expect

## 4. Hands-On Practice

### Exercise 1: Your First Conversation
1. Go to [claude.ai](https://claude.ai)
2. Ask Claude: "Explain HTML to me like I'm 17 and want to build my first website"
3. Read the response and ask follow-up questions

### Exercise 2: Generate Code
1. Ask Claude: "Write a simple HTML page that says 'Hello, I'm Omar' with a blue background"
2. Copy the code
3. Save it as `index.html` on your computer
4. Open it in your browser

### Exercise 3: Fix Code
Try this broken code and ask Claude to fix it:
```html
<html>
<head><title>My Page</head>
<body>
<h1>Welcome<h1>
<p>This is my site
</body>
```
Ask Claude: "This HTML has errors. Can you fix it and explain what was wrong?"

## 5. Key Takeaways
- Claude is a tool — you guide it with good prompts
- Always review the code Claude gives you — understand it, don't just copy it
- The more specific your prompt, the better the result
