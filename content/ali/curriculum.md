# Ali's Curriculum (Age 20)

## Goal
Build real-world AI-powered applications and understand the technology at a deeper level.

## Modules

### Module 1: AI Fundamentals
- How LLMs work (transformers, tokens, context windows)
- Prompt engineering — advanced techniques
- Understanding AI capabilities and limitations

### Module 2: Developer Tools
- Claude, GitHub Copilot, and AI-assisted coding
- Git & GitHub workflow (branches, PRs, collaboration)
- Terminal and command line proficiency

### Module 3: Full-Stack Web Development
- HTML, CSS, JavaScript foundations
- React or Next.js for frontend
- Node.js / API development
- Database basics (PostgreSQL, Supabase)

### Module 4: Building with AI APIs
- Claude API / Anthropic SDK
- Building chatbots and AI-powered features
- Prompt engineering for applications
- Handling tokens, costs, and rate limits

### Module 5: Deployment & DevOps
- Vercel, GitHub Actions, CI/CD basics
- Environment variables and secrets management
- Monitoring and debugging production apps

### Module 6: Real-World Project
- Plan, build, and deploy a complete AI-powered app
- Portfolio-worthy project
- Code review and best practices
